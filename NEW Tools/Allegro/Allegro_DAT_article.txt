== DAT == 
* ''' Format Type ''':    Archive <br> 
* ''' [http://en.wikipedia.org/wiki/Endianness Endian Order] ''': Big Endian <br>


=== Format Specifications === 

Allegro DAT files can be compressed or encrypted.<br>
Detailed description of the file format can be found in the [https://www.allegro.cc/manual/4/miscellaneous/datafile-format-information/ Allegro 4 documentation].<br>

=== Notes and Comments === 

* Archives used by allegro game engine.
* Passwords can be set in [https://www.allegro.cc/manual/4/tools/grabber/ Grabber tool] or in command-line DAT utility.
* To edit data in Grabber tool you need to type password with "-p" command into command-line <br>when launching Grabber, for example:<br> '''grabber.exe -phappy'''   // password "happy"
