FSB BATCH GENERATOR
It is a BAT script that generates FSB files from WAV files using FMOD Designer command line tool.


Requirements to use this tool:
- Line_Replacer.exe must be in the same directory
- fmod_designercl.exe must be in the same directory
- FMOD Designer project file  (.fdp) must be in the same directory
- "IN" and "OUT" directories must exist
